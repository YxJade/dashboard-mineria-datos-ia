# Dashboard de minería de datos con IA

Aplicación web desarrollada con Python y Flask capaz de analizar automáticamente archivos CSV, generar dashboards dinámicos y ejecutar modelos de Machine Learning sin modificar código.

## Funcionalidades

* Procesamiento automático de datasets CSV
* Generación dinámica de KPIs y gráficas
* Visualizaciones interactivas con Plotly
* Modelos de Machine Learning:

  * Regresión Lineal
  * K-Means + PCA
  * Naive Bayes
  * Árboles de Decisión
  * SVM (SVR)

## Tecnologías

* Python · Flask
* Pandas · Scikit-learn
* Plotly · JavaScript
* Anthropic SDK
* HTML · CSS · Vanilla JS

## Arquitectura

* Backend REST con Flask
* Procesamiento dinámico con Pandas
* Machine Learning con Scikit-learn
* Frontend interactivo con Plotly.js

## Características destacadas

* Adaptación automática a cualquier estructura CSV
* Interpretación automática de resultados mediante IA
* Dashboard universal orientado a múltiples dominios de datos

Por motivos de seguridad, el repositorio público contiene documentación, arquitectura y demostraciones visuales del sistema.
